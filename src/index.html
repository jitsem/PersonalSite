Who am i?

Hi, I'm Jitse: Industrial engineer in Electronics/ICT, currently working as a software engineer for AGFA NV., general tech-enthusiast, gamer, cat lover, Lord of the Rings fan & hobbyist beer-brewer. I made this site to play around a bit with various web-related technologies. Feel free to take a look around! Continue reading below if you want to know a bit more about me. My various contact details are listed in the footer of this page. If you encounter a bug on this site, feel free to submit an issue on Github

About this site


This site is written with help of BootStrap and is served via NGINX deployed in a Docker container. The docker container runs on a Raspberry PI 2 model B. Development is done mostly in Visual Studio Code and the entire project is stored on Github. The domain name "jitsem.be" is registered with TransIP and a dynamic DNS solution is provided by DuckDNS.
